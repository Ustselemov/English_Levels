# English Levels

This project was generated with the assistance of Codex AI and prompted by Ustselemov.

English Levels is a static web application that composes a character card from layered PNG assets and exports the result as a PNG image.

## Quick Start

1. Open `index.html` in a modern browser.
2. Adjust Listening, Pronunciation, Vocabulary, Grammar, and Fluency.
3. Click `Download PNG`.

## Repository Layout

- `index.html` - app entry page
- `styles.css` - layout and UI styles
- `app.js` - rendering and export logic
- `levels/` - source image assets
- `docs/` - project documentation
- `tests/` - testing notes
